# Sales Analysis Dashboard (Excel)

## 📊 Project Overview
This project is an interactive Excel dashboard built to analyze sales performance using Pivot Tables, charts, and slicers.

## 🔧 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers

## 📈 Key Insights
- Monthly sales trends analysis
- Category-wise sales distribution
- Region-wise performance
- Top 5 products by sales
